# Enumerable Methods

> Project to re-build a few built-in methods of Ruby Language.

![screenshot](./images/screenshot.png)
Methods based on Enumerable package.

## Requirements

<ul>
  <h3>Re-make this methods of enumerate:</h3>
  <li>#my_each</li>
  <li>#my_each_with_index</li>
  <li>#my_select</li>
  <li>#my_all?</li>
  <li>#my_any?</li>
  <li>#my_none?</li>
  <li>#my_count</li>
  <li>#my_map</li>
  <li>#my_inject</li>
